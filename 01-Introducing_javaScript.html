What is JavaScript?

JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997. ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.

JavaScript is a small ,lightweight,object-oriented scripting language that connects to objects in hosts enviroments and control them.

JS on the client-side accesses the Document Object Model to handle user interaction.

JS on the server-side can run servers and communicate to databases.



What is the difference between Js and Java?

Java compiles on systems of classes.

JS supports a system of small data types.

JavaScript is much more-free form,with less requirements.
